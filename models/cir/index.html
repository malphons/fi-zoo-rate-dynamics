Key Properties

Positive Rates Guaranteed

The σ√r diffusion term shrinks to zero as r → 0, preventing the rate from crossing zero. This is the key advantage over Vasicek.

Feller Condition

When 2ab ≥ σ², the drift dominates near zero, ensuring r(t) > 0 strictly. When violated, r can touch zero but reflects immediately.

Heteroskedastic Volatility

Effective volatility = σ√r. Higher rates produce wider path dispersion. This matches empirical observations in many markets.

CIR vs Vasicek Comparison

Same parameters applied to both models. Notice how Vasicek paths can go negative while CIR stays positive.

When Feller is Violated

If 2ab < σ², the rate can reach zero. At zero, the drift ab > 0 pushes it back up immediately. The process "reflects" off the boundary — it touches but never crosses zero.

CIR Bond Pricing

Like Vasicek, CIR has closed-form bond prices: P(t,T) = A(t,T)e^(-B(t,T)r) but with different A and B functions involving γ = √(a² + 2σ²).
