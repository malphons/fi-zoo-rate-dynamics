Vasicek Model

Equilibrium 1977
dr = a(b - r)dt + σdW
Ornstein-Uhlenbeck process for the short rate — Vasicek (1977)

Model Parameters

Key Properties

Mean Reversion

When r > b, the drift is negative, pulling rates down. When r < b, the drift is positive, pushing rates up. The parameter a controls the strength of this pull.

Negative Rates Possible

Unlike CIR, the Vasicek model uses constant volatility σ, so Gaussian shocks can push rates below zero. This is a known limitation (and feature for some markets).

Analytical Bond Pricing

Zero-coupon bond prices have a closed-form solution: P(t,T) = A(t,T)e^(-B(t,T)r(t)) making this model tractable for derivative pricing.

Understanding the Output

Expected Rate

E[r(t)] = b + (r0 - b)e^(-at)
The conditional mean reverts exponentially to b.

Rate Variance

Var[r(t)] = (sigma^2/2a)(1 - e^(-2at))
Variance increases initially and saturates at sigma^2/(2a).

Half-Life

t_1/2 = ln(2)/a
Time for the expected deviation from b to halve. A measure of reversion strength.
